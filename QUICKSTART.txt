🚀 БЫСТРЫЙ СТАРТ:

1. Дважды кликни START.bat
   ИЛИ
2. Открой index.html в браузере


📖 КАК ПОЛЬЗОВАТЬСЯ:

ЭТАП 1: Настройка (первый раз)
   ├─ Введи дату рождения Светы
   └─ Нажми "Начать магию"

ЭТАП 2: Добавление гостей
   ├─ Введи имя гостя
   ├─ Выбери дату рождения
   ├─ Нажми "Добавить" или Enter
   └─ Повторяй для всех гостей

ЭТАП 3: Магия совместимости
   ├─ Нажми "Начать расчёт совместимости"
   ├─ Смотри анимацию мандалы (3 сек)
   ├─ Изучай результаты:
   │  ├─ 💎 Индекс совместимости
   │  ├─ 📝 Сильные стороны
   │  ├─ ⚠️ Зоны внимания
   │  └─ 🌟 Космический совет
   ├─ Нажми "Показать магический факт"
   └─ Переходи к следующему гостю


✨ ЧТО РАССЧИТЫВАЕТСЯ:

• 🌠 Жизненный путь (35%) - нумерология
• ♈ Знаки зодиака (30%) - астрология
• 🔥 Элементы (20%) - стихии
• ⚡ Энергия (15%) - дополнительно


💜 РЕЗУЛЬТАТЫ:

80-100% = Космическая гармония! 💜
60-79%  = Прекрасный потенциал 💙
0-59%   = Связь через контрасты 💚


💡 СОВЕТЫ ДЛЯ ПРАЗДНИКА:

✓ Подключи к большому экрану/проектору
✓ Затемни комнату для эффекта неона
✓ Включи космическую музыку
✓ Показывай результаты как шоу
✓ Фотографируй реакции гостей!


🔒 КОНФИДЕНЦИАЛЬНОСТЬ:

✅ Все данные только на твоем компьютере
✅ Никаких отправок в интернет
✅ Работает полностью офлайн
✅ Можно удалить данные в любой момент


⚙️ НАСТРОЙКИ:

Файл: data/config.json
- Дата Светы
- Веса формулы
- Длительность анимации

Файл: data/texts.json
- Магические факты
- Шаблоны трактовок


🐛 ПРОБЛЕМЫ?

❌ Не открывается?
   → Попробуй другой браузер (Chrome/Edge)

❌ Анимация тормозит?
   → Уменьши durationMs в config.json

❌ Данные не сохраняются?
   → Используй локальный сервер (см. README.md)
